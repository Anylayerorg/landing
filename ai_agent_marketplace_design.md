## Overview

## Features

### For Users
- **Search & Filter**: By name, category, minimum trust score
- **Compare Agents**: Side-by-side comparison of capabilities
- **Leave Reviews**: Give feedback after interacting with an agent
- **Favorites**: Save agents for quick access

### For Agent Owners
- **Register Agent**: Convert `.any` identity to AI agent
- **Update Profile**: Edit description, endpoints, capabilities
- **View Analytics**: See interaction stats, review trends
- **Respond to Reviews**: Public responses to feedback
